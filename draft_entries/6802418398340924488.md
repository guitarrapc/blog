小ネタです。
Read-Hostでキーを読み取る時に、Caps Lockが有効化どうか判定したい場合にどうぞ。



## [Console]::CapsLock

まんまです。
面白くもなんともないので例で。

```ps1
Read-Host -Prompt "パスワードをどうぞ$(if([System.Console]::CapsLock){'(CapsLock が ONです。)'})"
```


実際CapsLockが有効でないとこうなります。

```ps1
パスワードをどうぞ:
```


CapsLockが有効だとこうなります。

```ps1
パスワードをどうぞ(CapsLock が ONです。):
```


以上でした。
