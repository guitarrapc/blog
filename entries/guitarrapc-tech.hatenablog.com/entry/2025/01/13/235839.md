PowerShell 7以降をscoopでインストールする方法を紹介します。

[:contents]

## PowerShellをインストール

scoopでPowerShellを追加します。PowerShell 7は`pwsh`という名前でMainバケットに登録されていいます。

```sh
#
$ scoop install pwsh
```


mainバケットはデフォルトで利用できますが、もし追加する場合は次のコマンドを実行してください。

$ scoop bucket add main
